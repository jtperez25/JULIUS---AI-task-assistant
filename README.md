# Meet JULIUS, your personal AI task assistant!

He can help you manage your week's schedule with a few simple commands:

  ADD EVENT: Add a new event to your schedule.
  
  REMOVE EVENT: Remove an existing event from your schedule.
  
  VIEW SCHEDULE: View all events scheduled for the week.
  
  HELP: Show this message.
  
  QUIT: Shut down JULIUS.
